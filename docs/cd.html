Linux Command: cd

Complete Beginner to Advanced Guide

üìå Overview

The cd (Change Directory) command is used to navigate between directories in Linux. It allows users to move through the file system efficiently.

Category: Navigation
Difficulty: Beginner

üßæ Syntax

cd [DIRECTORY]

If no directory is specified, the command moves to the user's home directory.

üü¢ Basic cd Commands (Beginner)

1. Go to a Specific Directory

cd Documents

Moves into the "Documents" directory inside the current folder.

2. Go Back to Home Directory

cd

Returns to the user's home directory.

3. Go to Parent Directory

cd ..

Moves one level up from the current directory.

4. Go to Root Directory

cd /

Navigates directly to the root (/) directory.

5. Go to Previous Directory

cd -

Switches back to the last visited directory.

üü° Intermediate Usage

Command Description
cd ~ Go to home directory
cd /home/user Navigate using absolute path
cd ../folder Move up one level and enter a folder
cd ./folder Enter a folder in the current directory
cd ../../ Move up two directory levels

üîµ Absolute vs Relative Paths

Absolute Path

cd /var/log

Starts from the root directory (/). Works from anywhere in the system.

Relative Path

cd logs

Starts from the current directory.

üî• Real-World Examples

Navigate to Downloads Folder

cd ~/Downloads

Navigate to Desktop

cd ~/Desktop

Move Back Two Directories

cd ../..

üß† Pro Tips for Beginners

  • Use pwd to check your current directory before using cd
  • Use Tab key for auto-completion of directory names
  • ~ represents the home directory
  • cd - is very useful for switching between two directories
