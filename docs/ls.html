Linux Command: ls

Complete Beginner to Advanced Guide

üìå Overview

The ls command is used to list files and directories in Linux. It is one of the most basic and important commands for navigating the file system.

Category: File Management
Difficulty: Beginner

üßæ Syntax

ls [OPTIONS] [DIRECTORY]

If no directory is specified, it lists files in the current directory.

üü¢ Basic ls Commands (Beginner)

1. List Files in Current Directory

ls

Displays all visible files and folders in the current directory.

2. Long Listing Format

ls -l

Shows detailed information like permissions, owner, size, and modification date.

3. Show Hidden Files

ls -a

Displays hidden files (files starting with a dot).

4. Long Format + Hidden Files (Most Used)

ls -la

Shows all files including hidden ones with detailed information.

5. Human Readable Sizes

ls -lh

Displays file sizes in KB, MB, or GB instead of bytes.

üü° Intermediate Options

Command Description
ls -R List files recursively in subdirectories
ls -t Sort files by modification time (newest first)
ls -S Sort files by size (largest first)
ls -r Reverse the sorting order
ls -1 Display one file per line
ls -d */ Show only directories

üîµ Advanced ls Options (Pro Level)

Command Use Case
ls -i Show inode numbers (useful in forensics)
ls -lu Show last access time
ls -lc Show last metadata change time
ls -ln Show numeric UID and GID instead of names
ls -F Display file type indicators (/ for directory, * for executable)
ls --full-time Show full timestamp with seconds

üî• Most Useful Real-World Combinations

Developer Favorite

ls -lah

Shows all files (including hidden) in long format with human-readable sizes.

Sort Files by Size (Readable)

ls -lhS

Detailed Recursive Listing

ls -lR

üìÅ Example with Directory Path

ls /home/user/Documents

Lists files inside a specific directory path.

üß† Pro Tips for Beginners

  • Hidden files in Linux start with a dot (.)
  • Use ls -la to see everything in a directory
  • Combine options like ls -lh for better readability
  • Great for cybersecurity, file auditing, and system navigation
